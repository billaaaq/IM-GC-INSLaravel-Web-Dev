Berlatih Fungsi

Contoh Soal

Buatlah sebuah fungsi bernama tampilkan() yang akan menampilkan "Aku Tampil" pada halaman HTML di elemen dengan ID contohJawaban.

Soal No.1

Buatlah sebuah fungsi bernama teriak(), yang mengembalikan nilai berupa "Halo Function!", kemudian tampilkan di elemen dengan ID jawaban1.

Soal No.2

Buatlah sebuah fungsi bernama kalikan(), yang menerima dua parameter, menghitung hasil perkalian kedua parameter tersebut, dan menampilkan hasilnya di elemen dengan ID jawaban2.
Contoh: kalikan(5, 4) akan menghasilkan output 20.

Soal No.3

Buatlah sebuah fungsi bernama buatKalimat(), yang menerima beberapa parameter seperti nama, umur, alamat, dan hobi, lalu menggabungkan semuanya menjadi sebuah kalimat lengkap.
Contoh: 

var name = "John Doe";
var age = 30;
var address = "Jln. Malioboro, Yogjakarta";
var hobby = "gaming";
buatKalimat(name, age, address, hobby);
Hasil yang ditampilkan: "Nama saya John Doe, umur saya 30 tahun, alamat saya di Jln. Malioboro, Yogjakarta, dan saya punya hobby yaitu gaming!"
